Implementation of {\ttfamily O\+C\+Client\+Response\+Handler} function type; called by stack on receipt of incoming {\ttfamily O\+C\+Client\+Response} from server. 

Converts the incoming response data to an {\ttfamily Observation} object, and then passes that to the {\ttfamily observe\+Behavior} method of the transaction\textquotesingle{}s {\ttfamily Co\+Service\+Provider} object for handling.

The {\ttfamily Co\+Service\+Provider} object is conveyed by the c\+\_\+\+Co\+SP parameter (in the C A\+PI, void$\ast$ context), which was passed (as the \char`\"{}context\char`\"{} pointer of the {\ttfamily O\+C\+Callback\+Data} param of {\ttfamily O\+C\+Do\+Resource}) by the {\ttfamily exhibit\+Stimulus} routine of the {\ttfamily Co\+Service\+Provider}.

I\+M\+P\+O\+R\+T\+A\+NT\+: this is an internal implementation routine, with no header prototoype. It serves as an intermediary between the stack the the Java application\textquotesingle{}s callback method.
