§ observe_behavior()

OCStackApplicationResult observe_behavior ( void *  c_CoSP,
OCDoHandle  c_TransactionHandle,
OCClientResponse *  c_OCClientResponse 
)

Wrapper for OCClientResponseHandler; called by stack on receipt of incoming OCClientResponse from server.

Convert the incoming response data to an ObservationIn object, and then pass that to the observeBehavior method of the transaction's CoServiceProvider object. That object is conveyed by the c_CoSP parameter (in the C API, void* context), which was passed (as the "context" pointer of the OCCallbackData param of OCDoResource) by the exhibitStimulus routine of the CoServiceProvider.

Parameters
[in]c_CoSPhandle to CoServiceProvider (client) object containing callback method; context in C API
[in]c_TransactionHandletoken identifying originating OCDoResource invocation (not the same as the CoAP token)
[in]c_OCClientResponseresponse data from server
Returns
result code of type OCStackApplicationResult
